Frequently Asked Questions

What is a CSS gradient?

A CSS gradient is a type of image that consists of a smooth transition between two or more specified colors. They are commonly used in web design for backgrounds and other decorative elements instead of using actual image files, as they load faster and are more flexible.

How do I use this gradient generator?

Simply choose between a linear or radial gradient, then use the controls to adjust the colors, their positions on the gradient, and (for linear gradients) the angle. The live preview and the CSS code will update instantly. You can add more colors using the 'Add Color' button.

What is the difference between linear and radial gradients?

A linear gradient transitions colors along a straight line. You can control the direction of this line using the angle slider. A radial gradient transitions colors outwards from a central point, like a circle.
